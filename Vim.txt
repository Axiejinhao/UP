vim -t tag

vim
末行模式命令:
保存 :w
退出,在保存为前提 :q
强行不保存退出 :q!
保存并退出 :wq / :x
浏览当前文件夹下的文件 :e .(所有) / 文件名
新建文件 :n 文件名
另存为一个新文件 :w 文件名

单行移动:
向后移动一个单词 w
向前移动一个单词 b
行首 0
行尾 $

行数移动:
文件顶部 gg
文件末尾 G
移动到数字对应行数 数字gg / 数字G / :数字

屏幕移动:
向上翻页 Ctrl+b
向下翻页 Ctrl+f
屏幕顶部 H
屏幕中间 M
屏幕底部 L

段落移动:
上一段 {
下一段 }

括号匹配及切换 %

标记 m[a~z] 
切换到标记的行 '[a~z]

选中文本:
从光标位置开始按照正常模式选择文本 v
选择光标经过的完整行 V
选择完整块 Ctrl+v

撤销和恢复撤销:
撤销上次命令 u
恢复撤销的命令 Ctrl+r

删除文本:
删除光标所在字符 x
删除移动命令选中的内容 d (d'x 从光标行删除到标记行)
删除光标所在的行 dd (ndd 连续删除n行)
删除至行尾 D

复制和粘贴:
复制移动命令选中内容 y
复制光标所在行 yy (nyy 连续复制n行)
粘贴 p

替换:
替换当前字符 r
替换当前行光标后的字符 R

缩进和重复执行:
在可视模式下缩进只需(> / <)
增加缩进 >>
减少缩进 <<
重复执行上次命令

查找:
查找str /str (n查找下一个 N查找上一个)
向后查找当前光标所在单词 *
向前查找当前光标所在单词 #

查找并替换(末行模式)
全局替换 :%s/旧/新/g
可视区域替换 :s/旧/新/g
确认替换 :%s/旧/新/gc

插入命令:
在当前行后/前插入一空行 o/O
在当前字符/行首插入文本 i/I

多行注释:
Ctrl+v进入可视块选择代码块,I进入编辑模式,在开头输入#

分屏命令:
横向分屏 :sp 文件名
纵向分屏 :vsp 文件名
切换到下一个窗口 w
